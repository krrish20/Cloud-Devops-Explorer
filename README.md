## ✨ Overview

The **Cloud & DevOps Explorer** is a single-page interactive dashboard that demystifies complex cloud concepts. By combining high-fidelity SVG animations with a gamified "Adventure Mode," it transforms dry documentation into an engaging learning journey.

## 🎯 Key Features

### 🕹️ Adventure Mode (Gamification)
*   **XP System**: Earn 10 XP for every module mastered.
*   **Leveling**: Progress from a **Novice** (Lvl 1) to a **Cloud Architect** (Lvl 10+).
*   **Skill Challenges**: Integrated quizzes for each concept to verify understanding.
*   **Progress Tracking**: Visual progress bars and checkmarks that persist via `localStorage`.

### 📊 High-Fidelity Visualizations
*   **Interactive SVGs**: Custom-built diagrams for Load Balancers, VPCs, Kubernetes, and more.
*   **Animated Flows**: Visual representations of data packets, CI/CD pipelines, and DevOps loops.
*   **Glow Engine**: Modern UI with glassmorphism and category-specific glow effects.

### 📚 Comprehensive Curriculum
*   **Foundations**: Cloud basics, IaaS/PaaS/SaaS, Regions & AZs.
*   **Networking**: VPCs, Subnets, DNS, CDNs, and Load Balancers.
*   **Containers & K8s**: Docker, Kubernetes Architecture, Service Mesh.
*   **Infrastructure as Code**: Terraform, Ansible, and Declarative configs.
*   **Modern Architecture**: Serverless (FaaS), Microservices, Event-Driven.
*   **Security & Ops**: IAM, Secrets Management, SRE, FinOps, and DR.

## 🛠️ Tech Stack

*   **Frontend**: Pure HTML5 / CSS3 (Vanilla)
*   **Styling**: Syne & JetBrains Mono typography, custom CSS Design System.
*   **Visuals**: Dynamic SVG Injection with CSS Keyframe Animations.
*   **Logic**: Vanilla JavaScript (ES6+).
*   **Persistence**: Browser `localStorage` API.
